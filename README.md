# Amazon Sales Analysis
### Tools: Power BI | SQL | DAX

## Project Overview
I developed this interactive report to transform raw Amazon transaction logs into strategic business intelligence. Using **SQL** for modeling and **DAX** for KPIs, I built visualizations to track YTD Sales and category-wise performance. This dashboard enables data-driven inventory management and identifies high-margin growth opportunities.

## Key Features & Insights
* **Sales Tracking**: Visualized total YTD Sales ($2.18M) and QTD Sales ($811.1K).
* **Category Performance**: Analyzed sales distribution across categories like Men's Shoes (43.18%), Cameras, and more.
* **Trend Analysis**: Monitored sales by month and weekly states to identify peak performance periods.
* **Product Reviews**: Tracked a total of 19.42M reviews to gauge customer engagement.

## Technical Skills Applied
* **SQL**: For data transformation and modeling.
* **DAX**: For calculating complex measures and KPIs.
* **Data Visualization**: Built interactive charts and filters for deep-dive analysis.
